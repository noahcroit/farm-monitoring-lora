## ESP32 Pinout for sensor node
#### LoRa Module RA-01
- MOSI -> 23
- MISO -> 19
- SCK -> 18
- CS -> 5
- RST -> 15
- IRQ -> 2

#### SHT31 Air temperature & humidity sensor
- SCL -> 22
- SDA -> 21

#### Soil Moisture Capacitive Sensor
- ADC -> 34

#### Serial Interface with Pi (Only for gateway device)
- RXD -> 16
- TXD -> 17
